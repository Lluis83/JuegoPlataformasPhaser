# Phaser Platform Game

A simple platform game built with Phaser 3.9.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Open your browser to `http://localhost:8080` and enjoy the game!

## Controls

- Left/Right Arrow: Move
- Up Arrow: Jump

## Troubleshooting

- If the game doesn't load, ensure Phaser is installed correctly.
- Check the console for any errors.
